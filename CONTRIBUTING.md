# Contributing to PinMaster

First off, thank you for considering contributing to PinMaster! It's people like you that make PinMaster such a great tool.

Following these guidelines helps to communicate that you respect the time of the developers managing and developing this open source project. In return, they should reciprocate that respect in addressing your issue, assessing changes, and helping you finalize your pull requests.

## ğŸ› Bug Reports

We use GitHub issues to track public bugs. If you encounter a problem with the extension:

1.  **Search** existing issues to see if the problem has already been reported.
2.  **Create a new issue** if one doesn't exist.
3.  **Provide context**:
    *   What browser and version are you using?
    *   What URL were you on?
    *   Steps to reproduce the error.
    *   Screenshots or console logs are highly encouraged.
